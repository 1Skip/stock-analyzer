# 股票分析系统 - 网页版

无需安装Python，可直接在浏览器中使用的网页版股票分析工具。

## 在线使用（无需安装）

### 方式一：Streamlit Cloud（推荐）

1. 访问: https://streamlit.io/cloud
2. 登录GitHub账号
3. 部署本应用即可在线访问

### 方式二：本地安装后使用

#### 步骤1: 安装Python
- 访问 https://www.python.org/downloads/
- 下载并安装 Python 3.9 或更高版本
- **重要**: 安装时勾选 "Add Python to PATH"

#### 步骤2: 安装依赖
打开命令提示符，运行：
```bash
cd C:\Users\skip8\stock_analyzer
pip install streamlit plotly yfinance pandas numpy requests akshare
```

#### 步骤3: 启动程序
```bash
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501

## 功能说明

### 1. 个股分析
- 输入股票代码（A股如: 000001, 600519）
- 选择市场（A股/美股/港股）
- 查看K线图、MACD、RSI、KDJ、布林带
- 获取综合交易建议

### 2. 热门股票
- 查看实时热门股票排行
- 涨幅榜、跌幅榜、成交量榜

### 3. 智能推荐
- 基于多因子技术分析
- 自动评分并推荐优质股票
