# 2026-06-16 更新记录

## 智能推荐策略复盘

- `RecommendationService` 新增只读策略复盘能力：从已保存的 T+1 计划历史中汇总策略、板块、计划数量、完成数量、待数据数量、1 日均收益和 1 日胜率。
- 策略复盘按最近交易日归档到 `recommendation_strategy_reviews` 本地缓存；读取复盘不会重新扫描股票池，也不会生成新的 T+1 推荐计划。
- 智能推荐页新增“最近交易日复盘”展示区和“刷新最近交易日复盘”按钮，展示复盘交易日、最近计划入场日、复盘概览、明日进化建议和历史计划。
- 明日进化建议仅来自历史计划表现的只读汇总，用于观察和人工参考；当前不会自动改变四个策略的股票池、过滤条件、评分、排序或推荐数量。

## 多因子稳健型轻筛修复

- 多因子稳健型轻筛阶段补充 K 线核心因子兜底：均线/量能、连续 3 日上涨、15 日内涨停触及三个 K 线核心因子全部不满足时，直接判为“K线核心因子不足”。
- 近 10 日涨幅超过 35% 的候选继续在轻筛阶段判为“非短期过热”。
- 本次调整用于避免明显不满足用户确认规则的候选进入深度检查；不新增消息催化、破板、收盘涨停、盘中追高或其他未要求条件。

## 验证
