# Bug fixes & context 2026-05-21

## 个股分析日K与同花顺口径修复

- A 股个股分析页的日K、成交量和技术指标继续固定使用 `1y` 前复权日K；实时行情只用于最新价、分时和成交量展示兜底，不合并进原始日K参与 MA/KDJ/BOLL/MACD 计算。
- 新增同花顺网页端日K源作为 A 股历史日K优先源，并保留东方财富、腾讯财经、新浪财经、Yahoo Finance 的真实数据回退链。
- 增加 A 股日K新鲜度检查：收盘后如果首选源最后一根日K不是当天，会继续尝试下一个真实日K源，避免页面缺少当天K线后显示上一交易日 MA。
- 个股分析页日K缓存加入分钟级版本键；页面 session 中已有的 A 股分析结果如果收盘后最后一根日K不是当天，会自动判为无效并重新分析，避免“成交量已是当天、日K和 MA 仍是上一交易日”的混搭。
- 万通发展 `600246` 加入静态名称兜底映射，避免中文名称解析失败时页面继续沿用旧标的。

## 图表和展示修复

- 个股分析页搜索输入、开始分析按钮和 Enter 提交共用同一提交逻辑；切页再返回时，输入框、当前标的、最新价和下方数据会绑定到同一个已分析标的。
- 日K与成交量拆成独立卡片；日K卡片右上角显示 MA5/MA10/MA20/MA30，成交量卡片右上角显示量、MA5、MA10、换手，并统一保留两位小数。
- 成交量图按真实成交量单位换算；同花顺/腾讯/新浪等返回“股”时转换为手，再展示“万手”；实时 quote 只在同一交易日用于覆盖成交量展示，不改写日K指标。
- K线图和 MACD/RSI/KDJ/BOLL 指标图隐藏底部日期标签；日K图关闭 Plotly 默认 range slider，避免底部出现一条被误认为指标的迷你K线预览。
- BOLL 图价格线改为深色主题下可见的浅色线。
